# s-button 说明文档
##功能

> 1、默认继承 el-button 所有属性
2、promise 防重复提交功能

## attributes

| 参数	 | 说明 | 类型	 | 可选值 | 默认值 |
| :-----| :---- | :---- | :---- | :---- |
| run | 快捷命令 | string |form.submit、form.search、form.reset、form.clear、form.close |- |
| confirm | 提示信息（会在点击按钮的时候在按钮附近弹出提示，确定后执行click事件） | string |- |- |
| dialogConfirm | 提示信息（会在点击按钮的时候居中弹窗提示，确定后执行click事件） | string |- |- |
